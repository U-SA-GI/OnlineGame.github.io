# Paiza Battle Ground

[使うのはJavaScriptとNode.jsだけ！30分で3Dオンライン対戦ゲームを作って公開してみた](https://paiza.hatenablog.com/entry/paizacloud_online_multiplayer_game)

# How to start

```
$ npm install
$ node server.js
```

or

```
$ npm install
$ nodemon server.js
```
